## 🔍 SEO Strategy

### On-Page
- Title: `[Keyword] + [ปี]`
- URL: `/blog/[slug]` — สั้น, มี keyword
- H1 = title, H2 = section headers (มี keyword)
- Meta description: 155 ตัวอักษร — มี keyword + CTA
- Internal links: 3-5 links ไปบทความที่เกี่ยวข้อง
- Alt text: ทุกรูปมี keyword

### Keywords Strategy
```
Short-tail:  automation, browser profile, multi account  (แข่งสูง — ใช้ใน homepage)
Mid-tail:    automation คือ, browser profile คือ, gemlogin สอนใช้  (เป้าหลัก)
Long-tail:   วิธีตั้ง line oa auto reply, tiktok shop automation affiliate ไทย  (เป้ารอง)
```
