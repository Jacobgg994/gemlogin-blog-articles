## 📊 Meta Business Suite — ศูนย์บัญชาการข้อมูล

### มันคืออะไร?

Meta Business Suite = แดชบอร์ดรวมทุกอย่าง — Facebook + Instagram + Messenger + Threads

เข้าได้ที่: **business.facebook.com** หรือแอป **Meta Business Suite** บนมือถือ

### หน้าตาหลักที่ต้องรู้

| แท็บ | ดูอะไร | ใช้ทำอะไร |
|------|--------|----------|
| **Home** | ภาพรวม — Reach, Engagement, Messages | เช็คเร็วๆ ทุกวัน |
| **Content** | ทุกโพสต์/Reels — เรียงตาม Performance | รู้ว่าโพสต์ไหนปัง |
| **Insights** | Audience, Trends, Benchmark | วางกลยุทธ์ |
| **Inbox** | Messages + Comments | ตอบลูกค้า |
| **Ads** | แคมเปญแอด | Optimize แอด |

## 🔢 10 ตัวเลขที่ต้องดู — และวิธีอ่าน

### 1. Reach — จำนวนคนที่เห็น

```
📊 ดูที่: Insights → Content → Reach

Reach สูง = Algorithm ดัน
Reach ต่ำ = Content ไม่ผ่านเกณฑ์ AI

✅ Reach > 30% ของ Followers = ดีมาก
⚠️ Reach 5-15% = ปกติ (2026 ค่าเฉลี่ย)
❌ Reach < 2% = มีปัญหา — เช็ค Quality Check
```

### 2. Engagement Rate — กี่% ของคนที่เห็น — มีส่วนร่วม

```
📊 สูตร: (Likes + Comments + Shares) ÷ Reach × 100

✅ > 5% = ปังมาก
⚠️ 0.5-5% = ปกติ
❌ < 0.5% = Content ไม่ resonate

💡 Engagement Rate > Reach — เพราะ 1 คนอาจ engage หลายครั้ง
   ดูที่ "Engaged Users" แทน "Total Engagement"
```

### 3. Comments — วัด Deep Engagement

```
📊 ดูที่: Content → เลือกโพสต์ → Comments

✅ คอมเมนต์ 5+ คำ > 50% ของคอมเมนต์ทั้งหมด = Good
⚠️ คอมเมนต์อิโมจิ 1-2 คำ > 80% = ผิวเผิน

💡 Algorithm 2026 ให้น้ำหนักกับ:
   - คอมเมนต์ 5+ คำ = 3x ของ Like
   - Reply จาก Page = 2x
   - คอมเมนต์จากเพื่อนของคนโพสต์ = 1.5x
```

### 4. Retention Rate (Reels) — % คนดูผ่านครึ่ง

```
📊 ดูที่: Content → Reels → Retention Graph

✅ > 50% Retention ที่ 50% ของคลิป = Algorithm ดันต่อ
⚠️ 30-50% = ปรับปรุง Hook
❌ < 30% = เปลี่ยน 3 วิแรกใหม่

💡 กฎ 50%: ถ้า retention ผ่านครึ่งคลิป → Reach พุ่ง 4 เท่า
```

### 5. Follower Growth — โตแบบสุขภาพดี

```
📊 ดูที่: Insights → Followers → Net Growth

✅ โต > 5% ต่อเดือน = Healthy
⚠️ 1-5% = โตช้าแต่ไปต่อ
❌ ติดลบ = Content ไม่โดน หรือ Followers คุณภาพต่ำ

ดูแยก:
- Organic Followers (ฟรี)
- Paid Followers (จากแอด)
- Unfollows (เสียไปกี่คน)
```

### 6. Best Posting Time — เวลาไหนดี

```
📊 ดูที่: Insights → Audience → When Your Followers Are Online

✅ โพสต์ 30 นาที ก่อนช่วง Peak — ให้ Algorithm มีเวลา Index
⚠️ โพสต์ช่วง Peak พอดี — แข่งกันเยอะ

💡 ข้อมูลเฉลี่ยไทย:
   - จ-ศ: 19:00-21:00 (Peak)
   - ส-อา: 12:00-14:00 และ 20:00-22:00
```

### 7. Audience Demographics — ใครคือคนดู

```
📊 ดูที่: Insights → Audience → Demographics

ถามตัวเอง:
- คนดูตรงกับกลุ่มเป้าหมายเราไหม?
- ผู้หญิง 80% — แต่ขายของผู้ชาย → ปรับ Content
- อายุ 18-24 — แต่สินค้าราคาแพง → กลุ่มไม่มีกำลังซื้อ
```

### 8. Video Views vs Completion Rate

```
📊 ดูที่: Content → Videos → Video Performance

Views สูง + Completion ต่ำ = Hook ดี แต่เนื้อหาไม่ส่งมอบ
Views ต่ำ + Completion สูง = เนื้อหาดี แต่ Hook อ่อน

💡 3-Second View ÷ Total Views = Hook Rate
   > 70% = Hook ดี
   < 50% = ต้องเปลี่ยน 3 วิแรก
```

### 9. CTA Click-Through Rate (CTR)

```
📊 ดูที่: Content → เลือกโพสต์ → Link Clicks ÷ Reach

✅ > 3% = CTA แรง
⚠️ 1-3% = ปกติ
❌ < 1% = CTA ไม่ชัด หรือ Audience ไม่ใช่

💡 คนไทยคลิกอะไร:
   - "ดาวน์โหลดฟรี" > "ดูเพิ่มเติม"
   - "แชทเลย" > "ติดต่อเรา"
   - "รับโปร" > "สมัครสมาชิก"
```

### 10. Net Promoter Score (NPS) — วัดความรัก

```
📊 วิธีทำ: ทำให้เอง

1. เลือกลูกค้า 20-30 คนที่ Active
2. ถาม: "0-10 — คุณจะแนะนำเพจเราให้เพื่อนไหม?"
3. คำนวณ:
   - 9-10 = Promoter
   - 7-8 = Neutral
   - 0-6 = Detractor
   NPS = %Promoters - %Detractors

✅ > 50 = Excellent
⚠️ 0-50 = Good
❌ < 0 = มีปัญหา
```

## 🔄 วงจร Data-Driven — ทำซ้ำทุกสัปดาห์

### 1. รวบรวมข้อมูล (ทุกวันจันทร์)

```
📊 Export จาก Meta Business Suite:
- Content tab → Export Data → เลือก 7 วันล่าสุด
- เปิดใน Google Sheets
- เพิ่มคอลัมน์: Engagement Rate, CTR, Comment Quality
```

### 2. วิเคราะห์ — ถาม 3 คำถาม

```
❓ "โพสต์ไหน Engagement Rate สูงสุด?"
   → ทำแนวนี้มากขึ้น

❓ "โพสต์ไหน Reach สูงสุด?"
   → วิเคราะห์ว่า Algorithm ดันเพราะอะไร — Hook? Topic? Format?

❓ "โพสต์ไหน Engagement ต่ำสุด?"
   → หยุดทำแนวนี้ — หรือทดลองเปลี่ยน Hook
```

### 3. ตั้งสมมติฐาน — ทดสอบ

```
📝 "ถ้าเราเปลี่ยน Hook จาก 'ขาย' เป็น 'สอน' 
    Reach จะเพิ่มขึ้น 20%"

→ ทดสอบ 1 สัปดาห์
→ วัดผล
→ ถ้าจริง = เปลี่ยนทั้งเพจ
→ ถ้าไม่จริง = เปลี่ยนสมมติฐานใหม่
```

### 4. Repeat — Every Week

```
Week 1: วัด Baseline
Week 2: ทดสอบ 1 ตัวแปร
Week 3: วัดผล — เก็บที่เวิร์ค ทิ้งที่ไม่เวิร์ค
Week 4: ทดสอบตัวแปรใหม่
...
Week 52: คุณมี Data-Driven Playbook ของตัวเอง
```

## 🧪 ทดลอง A/B Test แบบง่ายๆ

### วิธีทำโดยไม่ต้องใช้ Ads

```
📝 Organic A/B Testing:

1. เลือก 1 ตัวแปรที่จะทดสอบ:
   - Hook (คำถาม vs สถิติ)
   - Format (Reels vs รูป)
   - เวลาโพสต์ (เช้า vs เย็น)
   - CTA (แชท vs คอมเมนต์)

2. โพสต์ Version A — วันจันทร์
3. โพสต์ Version B — วันพุธ (เนื้อหาเดียวกัน เปลี่ยนแค่ 1 อย่าง)
4. วัดผลวันศุกร์ — อันไหน Engagement Rate สูงกว่า = ใช้ต่อไป
```
