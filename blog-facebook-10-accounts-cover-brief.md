# Cover Design Brief — "วิธีเปิด Facebook 10 บัญชีใน 15 นาที (ไม่แบน!) ด้วย GemLogin"

---

## 1. ขนาดปกตามแพลตฟอร์ม

| Platform | Size (px) | Aspect Ratio | หมายเหตุ |
|---|---|---|---|
| **Facebook Feed / Post** | 1200 x 630 | 1.91:1 | ขนาดหลักที่ใช้ออกแบบ |
| **Blog Thumbnail** | 1280 x 720 | 16:9 | สำหรับ SEO / Open Graph |
| **LINE OA Rich Message** | 1200 x 600 | 2:1 | ครอบตัดจาก 1200x630 ได้ |
| **TikTok Video Cover** | 1080 x 1920 | 9:16 | แนวตั้ง ต้อง redesign layout |

> **แนะนำ**: ออกแบบที่ 1200x630 px ก่อน แล้วปรับขยาย/ครอบตัดไปแพลตฟอร์มอื่น

---

## 2. โทนสี (Brand Palette)

| Element | Color Code | ใช้กับ |
|---|---|---|
| **Primary — เขียวเข้ม** | `#0d9488` (Teal 600) | พาดหัว, ปุ่ม CTA, กรอบตัวเลข, Accent |
| **Dark — ดำสนิท** | `#0a0a0a` (Neutral 950) | พื้นหลัง, ข้อความหลัก |
| **Light — ขาวสะอาด** | `#fafafa` (Neutral 50) | พื้นหลังฝั่งซ้าย (ถ้าใช้ Split Layout), ข้อความบนพื้นเข้ม |
| **Accent — เขียวอ่อน** | `#5eead4` (Teal 200) | Highlight, ตัวเลขรอง, เส้นแบ่ง |
| **Warning — ส้ม/แดง** | `#ef4444` (Red 500) | ใช้เฉพาะ "ปัญหา" ในแนวทาง Problem/Solution |

### Gradient Option
```
พื้นหลัง Gradient: จาก #0a0a0a → #0d9488 (ซ้าย → ขวา หรือ บน → ล่าง)
ใช้เมื่อต้องการให้ปกดูมีมิติ ไม่แบน
```

## 3. Typography (Font)

| Usage | Font | Weight | ขนาด (1200x630) |
|---|---|---|---|
| **พาดหัวหลัก (Headline)** | Kanit / Noto Sans Thai | Bold (700) | 48-64px |
| **ตัวเลขเด่น** | Kanit / Sarabun | ExtraBold (800) | 72-96px |
| **ข้อความรอง (Subhead)** | Noto Sans Thai | Medium (500) | 20-28px |
| **คำโปรย / รายละเอียด** | Noto Sans Thai | Regular (400) | 14-18px |
| **ข้อความเล็ก / โลโก้ (Footer)** | Noto Sans Thai | Light (300) | 12-14px |

> **Tip**: Kanit ให้ฟีลโมเดิร์น คม เท่ เหมาะกับพาดหัวสั้นๆ
> Noto Sans Thai อ่านง่าย สบายตา ใช้กับข้อความยาว

## 4. องค์ประกอบที่ต้องมี

| # | Element | รายละเอียด | ตำแหน่ง |
|---|---|---|---|
| 1 | **พาดหัวสั้น** | 3-5 คำ เช่น "ไม่แบน ไม่จำกัด" หรือ "10 บัญชี 15 นาที" | กลาง-บน, เด่นที่สุด |
| 2 | **ตัวเลขเด่น** | "10" และ "15" — ใช้ตัวเลขขนาดใหญ่ ดึงดูดสายตา | กลางภาพ, ใกล้พาดหัว |
| 3 | **ภาพประกอบ** | Browser icons / มือถือ 3D / หน้าจอ GemLogin (แบบง่าย) | ด้านล่างของพาดหัว หรือ flanking ตัวเลข |
| 4 | **GemLogin Logo** | โลโก้ GemLogin หรือคำว่า "GemLogin" | มุมล่างขวา หรือล่างซ้าย |
| 5 | **Micro-text** | เช่น "ใช้คอมเครื่องเดียว" / "No ban guarantee" / "Facebook Approved" | ข้างหรือใต้พาดหัว |

### องค์ประกอบเสริม (Nice to have)
- เส้นกรอบ/เส้นคั่น ที่เป็นลูกคลื่น — สื่อถึง "Ocean / Kvasir" theme
- สัญลักษณ์ ✅ ข้างตัวเลขเพื่อแสดงความสำเร็จ
- Badge "New" หรือ "2026" ที่มุมบน

---

## 5. องค์ประกอบที่ห้ามมี (Avoid)

| ❌ | เหตุผล |
|---|---|
| **รูปหน้าคน (stock photo)** | ดูไม่เป็นทางการ คล้ายสแปม |
| **ข้อความย่อหน้าสั้นเกินไป** | อ่านบนมือถือไม่รู้เรื่อง |
| **พื้นหลังรก / texture เยอะ** | บดบังพาดหัว |
| **โลโก้ Facebook ทางการ** | เสี่ยงละเมิด trademark — ใช้แค่ข้อความ "Facebook" |
| **ใช้สีเกิน 4 สี** | ดูรก ไม่เป็นมืออาชีพ |
| **ข้อความบนพื้น Gradient ที่อ่านยาก** | ต้องตรวจ Contrast ก่อน Final |
| **รูป screenshot จริงของ GemLogin (ย่อเล็ก)** | ดูเละ ไม่คม — ใช้ illustration แทน |

## 6. ตัวอย่าง Layout: 3 แนวทาง

---

### แนวทาง A — MINIMAL (สะอาด ดูแพง เน้นพรีเมียม)

```
┌────────────────────────────────────────────┐
│  [Badge: 2026]                             │
│                                            │
│            เปิด 10 บัญชี                    │
│             15 นาที                        │
│                                            │
│         ┌──────┐   ┌──────┐               │
│         │ 10   │   │ 15   │               │
│         │ บัญชี│   │ นาที │               │
│         └──────┘   └──────┘               │
│                                            │
│          ใช้คอมแค่เครื่องเดียว              │
│                                            │
│    ─────────────────────────────           │
│    💻 ภาพ Browser Icon แบบ Minimal        │
│                                            │
│                             [GemLogin]     │
└────────────────────────────────────────────┘
```

- **พื้นหลัง**: #0a0a0a (ดำ) หรือ Gradient ดำ → เขียวเข้ม
- **พาดหัว**: Kanit Bold #fafafa
- **ตัวเลข**: Kanit ExtraBold #5eead4 (เขียวอ่อน) ขนาด 80px+
- **ความรู้สึก**: Modern, Tech, Premium — เหมาะกับ Facebook Post และ Blog

---

### แนวทาง B — BOLD / ตัวเลขเด่น (สะดุดตา โฟกัสที่ Performance)

```
┌────────────────────────────────────────────┐
│                                            │
│                                            │
│              🚀                            │
│           10                                 │
│         FACEBOOK                          │
│          บัญชี                             │
│                                            │
│         ⏱ 15 นาที • 💻 เครื่องเดียว        │
│                                            │
│   ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─       │
│   👉 เปิดได้เลยวันนี้ ด้วย GemLogin         │
│                                            │
│   ┌─────────────────────────────────┐      │
│   │  ใช้ Proxy + Fingerprint แยก   │      │
│   │  Facebook มองว่าคนละคนกัน      │      │
│   └─────────────────────────────────┘      │
│                                            │
│                          [GemLogin Logo]   │
└────────────────────────────────────────────┘
```

- **พื้นหลัง**: #0a0a0a
- **ตัวเลข "10"**: เต็มพื้นที่เกือบครึ่งภาพ ขนาด 120px+ สี #0d9488 (เขียว) วางตรงกลาง
- **ข้อความ "FACEBOOK บัญชี"**: ข้างหรือใต้เลข 10
- **บรรทัด "15 นาที • เครื่องเดียว"**: ใต้พาดหัว ขนาด 24px #fafafa
- **ความรู้สึก**: Powerful, Result-driven, Action — ดึงดูดคนที่ต้องการผลลัพธ์เร็ว
- **ที่มา**: ได้แรงบันดาลใจจาก Xpiral design ที่ใช้ตัวเลขเป็นพระเอก เน้นความสำเร็จ measurable

---

### แนวทาง C — PROBLEM / SOLUTION (ก่อน-หลัง สร้าง Contrast)

```
┌────────────────────────────────────────────┐
│                                            │
│   😫 BEFORE              🚀 AFTER         │
│   ┌──────────┐          ┌──────────┐      │
│   │ ❌ แบน   │          │ ✅ 10 บ.  │      │
│   │ ❌ เสีย   │          │ ✅ 15 น.  │      │
│   │   เวลา    │          │ ✅ ไม่แบน │      │
│   │ ❌ เสีย   │          │          │      │
│   │   เงิน    │          │          │      │
│   └──────────┘          └──────────┘      │
│                                            │
│        เปลี่ยนจากแบน → ขายได้วันละ万        │
│              ด้วย GemLogin                 │
│                                            │
│              [GemLogin Logo]               │
└────────────────────────────────────────────┘
```

- **พื้นหลัง**: Split Layout — ซ้าย #a60000 (แดงเข้ม) / ขวา #0d9488 (เขียว)
  หรือซ้ายดำ / ขวาเขียว
- **BEFORE**: สีแดง #ef4444, Icon ❌, ข้อความสั้นๆ (แบน, เสียเวลา, เสียเงิน)
- **AFTER**: สีเขียว #0d9488, Icon ✅, ข้อความ (10 บัญชี, 15 นาที, ไม่แบน)
- **ตรงกลาง**: Icon ลูกศร ➡️ หรือเส้น Transition
- **พาดหัวล่าง**: บอกว่าการเปลี่ยนผ่านเกิดขึ้นด้วย GemLogin
- **ความรู้สึก**: Emotional contrast — เห็นใจก่อน (BEFORE) → โล่งใจ (AFTER)
- **ใช้กับ**: Facebook Ad และ LINE OA Rich Message ได้ดี เพราะเล่าเรื่องในภาพเดียว

## 7. สรุปการเลือกแนวทางตามวัตถุประสงค์

| ถ้าต้องการ... | ใช้แนวทาง |
|---|---|
| ดูพรีเมียม Professional | **A — Minimal** |
| ดึงดูดคนสนใจ Performance / ผลลัพธ์ | **B — Bold/ตัวเลขเด่น** |
| สร้าง Conversion สูง (Ad / LINE) | **C — Problem/Solution** |
| Blog Thumbnail / SEO | **A หรือ B** |
| Facebook Post | **B หรือ C** |
| TikTok Cover (9:16) | **B** (ตัวเลขเด่นเห็นง่ายบนมือถือ) |

## 8. ข้อแนะนำเพิ่มเติมสำหรับ Designer

- **Export**: PNG สำหรับ Social Media, WebP (คุณภาพ 80%) สำหรับ Blog
- **Safe Zone**: เนื้อหาสำคัญต้องอยู่ภายใน 1140x570 px (เผื่อ crop)
- **Mobile**: 1200x630 จะถูกย่อเหลือ ~470x247 บน Facebook Feed — ทดสอบว่า "10" และ "15" ยังชัด
- **Text Overlay**: ถ้าต้องทำ Version ไม่มีข้อความบนภาพ (สำหรับ Ad Platform ที่ไม่อนุญาต) — ใช้ Icon-based layout
- **A/B Test Layout**: แนะนำเริ่มที่แนวทาง B (ตัวเลขเด่น) เพราะขาย Performance ได้ดีกับกลุ่มเป้าหมายคนทำการตลาดออนไลน์
