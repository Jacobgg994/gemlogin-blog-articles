## REVEAL — สิ่งที่ Facebook ไม่ได้บอกคุณ

รู้ไหมว่า **96% ของเว็บไซต์ดังๆ** — Facebook, Google, TikTok, Shopee — ใช้เทคโนโลยีที่เรียกว่า **Browser Fingerprint Tracking**

พูดง่ายๆ: ทุกครั้งที่คุณเปิดเว็บ Browser จะส่งข้อมูลเฉพาะตัวของคุณหลายสิบอย่างให้เว็บนั้นแบบอัตโนมัติ

ต่อให้คุณ *ไม่ได้ Login* ด้วยซ้ำ งานเข้าเลยใช่ไหมล่ะ?

```
🖥️ IP Address — คุณอยู่ประเทศอะไร ใช้ ISP ไหน
🎨 Canvas Fingerprint — การ์ดจอคุณวาดภาพยังไง
🎮 WebGL — GPU ตอบสนองยังไงบ้าง
🎵 AudioContext — คลื่นเสียงที่เครื่องคุณผลิต
🔤 Fonts — คุณลงฟอนต์อะไรไว้บ้าง
🌐 UserAgent — Chrome หรือ Safari? เวอร์ชั่นไหน?
🕐 Timezone — คุณตั้งเวลาประเทศอะไร
🔠 Language — ภาษาไหนคือภาษาหลักของคุณ
📐 Screen Resolution — จอคุณกว้างเท่าไหร่
```

รวมข้อมูลพวกนี้เข้าด้วยกัน = **Digital Fingerprint** ซึ่งมีโอกาส uniqueness สูงมาก — เฉพาะเจาะจงระดับ **"คือคุณคนนี้แน่นอน"**

**แล้ว Facebook เอาข้อมูลนี้ไปใช้ยังไง?**

ลองดู:
- บัญชี A ใช้ชื่อ "สมชาย" — อีเมล X, เบอร์ X
- บัญชี B ใช้ชื่อ "ไมเคิล" — อีเมล Y, เบอร์ Y
- แต่เปิดจาก Browser **เครื่องเดียวกัน**

→ Facebook Match Fingerprint → **"คนนี้แหละคนเดิม"** → BAN ทั้งสองบัญชี 🔨

**และนี่คือสาเหตุที่ Appeal ไม่ผ่าน** — เพราะคุณบอกว่า "ไม่ได้ทำผิด" แต่ระบบมันเห็นว่า "คนๆ เดียวกันมีหลายบัญชี" ซึ่ง Facebook ถือว่าผิด

## Browser ปกติ กับ Antidetect Browser ต่างกันยังไง?

ลองนึกภาพตาม:

🔴 **Browser ปกติ (Chrome, Edge, Safari)**
เหมือน **บัตรประชาชนใบเดียว** — ใช้เปิดประตูที่ไหน ก็มีคนจดจำคุณได้ทุกที่

🟢 **Antidetect Browser**
เหมือน **ชุดพรางตัว** หรือ **หน้ากาก** — แต่ละครั้งที่เปิดเว็บ คุณจะดูเหมือน "คนละคน" กับครั้งก่อน

| Browser ปกติ | Antidetect Browser |
|---|---|
| ส่ง Fingerprint จริงของคุณตลอด | ปลอม/เปลี่ยน Fingerprint ได้ |
| เว็บรู้ว่าคุณคือคนเดิม | เว็บคิดว่าคุณคือคนละคน |
| ใช้หลายบัญชี = เสี่ยงโดนลิงก์ | ใช้กี่บัญชีก็แยกกันขาด |
| โดนแบน = แบนทั้งเครื่อง | โดนแบนแค่ Profile นั้น |
| ไม่ต้องตั้งค่า (แต่ป้องกันอะไรไม่ได้) | สร้าง Profile แล้วใช้เลย |
